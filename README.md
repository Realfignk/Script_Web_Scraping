# Script Web Scrapping

Script para obtener los comentarios y direcciones de correo electrónico.

##  Características
-  Portable
-  Ejecución rápida.
-  Guarda los resultados en un archivo de texto.

## Requisitos
Es necesario tener instalado o contar con:
* Python 3
* Librerias bs4 y request.

## Instalación y Uso

1. Copia el script o extrae el archivo .zip.
2. Ejecuta el script: python3 nombre_script.py
3. Ingresa la url cuando se solicite.
